# Bezier-BSpline-Surface-Modeling
SW40503 Mini Project

## Motivation
------------------
To get exposure on constructing 3D object using Bezier and B-Spline surface function. <br />


## Features
------------------
- Interactive tools
- Mapping photographic images to the surface
- Surface explosion
